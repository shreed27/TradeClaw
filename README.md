# FinClaw — Mastermind Trading Brain

Indian stock index options A2A multi-agent system.
Harshad Mehta as Boss. Eight legendary traders as sub-agents.

- A2A message bus (broadcast + DM)
- Persona-driven agents (markdown configs)
- Multi-round deliberation + voting
- Helm orchestrator: consolidates voices into final trade
- Self-evolving learning loop (weight adjustments by win-rate)
- Mock market feed + option strategy library

Run `python main.py` from the repo root.
